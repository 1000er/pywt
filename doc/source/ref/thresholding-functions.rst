Thresholding functions
======================

The :mod:`~pywt.thresholding` helper module implements the most popular signal
thresholding functions. See `plot_thresholds.py` in the `demo`_ directory for
an illustration of non-negative garotte and firm thresholding in comparison to
soft and hard thresholding.
